----------------------------------------------------------------------------
I. УСТАНОВКА
----------------------------------------------------------------------------
1) Перед копированием файлов из папки /install переименуйте файл 
protected\modules\admin\controllers\__AdminController.php
в необходимое, напр., "MyExampleController.php".
* eсли требуется измените в PHP-коде файла имя класса контроллера.

2) Скопируйте файлы из папки /install в корень сайта.
