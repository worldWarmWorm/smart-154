Инструкция по установке и использованию модуля
----------------------------------------------------------------------------
Содержание:
I. УСТАНОВКА
II. ИСПОЛЬЗОВАНИЕ

----------------------------------------------------------------------------
I. УСТАНОВКА
----------------------------------------------------------------------------
1) Перед копированием файлов из папки /install переименуйте файл 
protected\modules\admin\controllers\__AdminController.php
в необходимое, напр., "MyExampleController.php".
* eсли требуется измените в PHP-коде файла имя класса контроллера.

2) Скопируйте файлы из папки /install в корень сайта.

----------------------------------------------------------------------------
II. ИСПОЛЬЗОВАНИЕ
----------------------------------------------------------------------------
1) ПУБЛИКАЦИЯ ТЭГА "CANONICAL" для страниц с параметрами в строке или страниц пагинации с параметрами.
Для прямых страниц пагинации (http://example.com/link?p=2) публикация тэга происходить не будет.
Для ссылки http://example.com/link?p=1 будет опубликован тэг "canonical" http://example.com/link.
<? use seo\components\helpers\HSeo; ?>
<? HSeo::canonicalByGET(['p'=>true, 'page'=>true], ['p'=>[1], 'page'=>[1]]); ?> 


Публикация тэга каноникал для товара, страница которого доступна по нескольким URL
if(Y::isAction($this, 'shop', 'product')) {
	$url=Y::createUrl('shop/product', ['id'=>$_GET['id']]);
	if($url != ('/'.Y::request()->pathInfo)) {
		HSeo::canonicalByUrl('shop/product', ['id'=>$_GET['id']]);
	}
}
else HSeo::canonicalByGET(['p'=>true, 'page'=>true], ['p'=>[1], 'page'=>[1]]); 

2) РЕГИСТРАЦИЯ ТЭГА ПОСТРАНИЧНОЙ НАВИГАЦИИ (по рекомендации Google)
HSeo::registerLinkTagNav($dataProvider, 'mycontroller/myaction', ['id'=>$model->id]);
