XSL Stylesheets for XML Sitemaps
==
This package contains 3 XSL stylesheets, for normal, news, and video sitemaps, 
to be used for styling and debugging XML sitemaps. For more info, check out:

http://yoast.com/xsl-stylesheet-xml-sitemap/

To use them, add a line like this to your XML Sitemap output:

<?xml-stylesheet type="text/xsl" href="/url/to/xsl-stylesheet.xsl"?>
